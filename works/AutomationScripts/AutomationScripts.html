Automation Scripts

  • Category: WebDesign
  • Team Size: 1
  • Year: 2024

As part of the final project (PAP) for the Technical Course in Computer Systems, this project was developed in two parts. The first part involves the development of scripts to automate repetitive tasks within the video game OLD Runescape. The second part involves creating a website for the dissemination and installation of these scripts.

The choice of theme was primarily influenced by two factors. The first was related to the fact that among the themes of the final projects in recent years, none seemed interesting to me. Therefore, I proposed the theme of scripts to the supervising professor, which was later accepted because it was a different idea from previous projects or those being proposed. The second factor was my need to feel that I was doing something from which I would learn something new.

Script Features

The scripts developed for OLD Runescape offer a variety of features to automate repetitive tasks:

  • Resource Gathering Automation: Scripts to automatically gather wood, ores, and other resources.
  • Combat Automation: Scripts to train combat skills by automatically defeating monsters.
  • Skill Training Automation: Scripts to train skills such as fishing, cooking, and crafting.

Design Process

The design process for the website to disseminate and install the scripts involved several stages:

  1. Research: Understanding user needs and best design practices for script websites.
  2. Wireframing: Creating wireframes to outline the structure and layout of the website.
  3. Design: Developing the visual design, including color schemes, typography, and icons.
  4. Development: Building the website using HTML, CSS, and JavaScript.
  5. Testing: Conducting usability tests to gather feedback and make improvements.

Stages of the design process.

Community Engagement

Engaging the community is a key aspect of the automation scripts project. Here are some strategies used to foster engagement:

  • Featured Content: Highlighting popular discussions and news updates.
  • Feedback Mechanism: Providing a platform for users to give feedback and suggest improvements.

Future Plans
